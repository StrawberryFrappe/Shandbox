# Shandbox

Shandbox is a hand-tracking physics sandbox for tinkering with cellular automata and computer vision.

## Interaction
- **Sand**: Pinch index and thumb to spawn.
- **Fire**: Touch pinky to thumb.
- **Water**: Cup palm and point thumb down.
- **Eraser**: Madankosappo hand sign (index and middle extended).
- **Physics**: Move your hand to displace particles.

## Technology
- p5.js and ml5.js (MediaPipe Handpose).
- Web Workers for background inference.
- Custom particle physics engine.

## Usage
Open `index.html` via local server or HTTPS for camera access. Alternatively, view on GitHub Pages.
